Dataset Overview

The OmniIHC2025 dataset encompasses IHC data from more than 350 patients across 19 staining types and collected from 5 cohorts, including CD11b, CD11c, CD14, CD15, CD16, CD33, CD66b, CD68, CD73, CD86, CD163, CD206, IDO, LAG3, PD-1, Siglec-15, SIRPa, TIM3, and Ki67.

Evaluation Metrics

Panoptic Quality (PQ)
Measures the classification accuracy of pixel points, reflecting the correctness of instance segmentation.
Mean Absolute Error (MAE)
Computes the absolute error between predicted and true cell counts, as well as the Tumor Proportion Score (TPS).

Challenge Rules

  • Participants must use only the provided dataset for training
  • All submissions must be fully automatic methods
  • Teams must submit their Docker container along with a short paper
  • Final results will be evaluated on the test set
