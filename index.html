What is OmniIHC?

The OmniIHC2025 Challenge presents a comprehensive dataset for tumor profiling across multiple immunohistochemistry stained images.

Challenge Features:

  • ✓ Over 1 million cell annotations
  • ✓ Data from 350+ patients
  • ✓ 19 diverse IHC staining types
  • ✓ Collected from 5 cohorts
  • ✓ High-quality cell-level annotations
  • ✓ Multiple biomarkers analysis
  • ✓ Cell segmentation task
  • ✓ Cell quantification task
